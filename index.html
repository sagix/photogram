Setup directory

You must put all your images in the same directory.

And add a CSV named data.csv in this directory.

With comma as field delimiter.

And double quotes as text delimiter

empty lines or with more then two columns are ignored

Example

in a file sheet:

image name legend of photogram
1 action for scene one
2 two two
3.1 start of "traveling"

in the CSV exported

1,action for scene one
2,two two
3.1,"start of ""traveling"""

Auto save

At the first opening and if you do not edit any label, the data.csv file is read.

As soon as you start editing, auto-save is on and start saving data int the save.json file.

After that, at each opening, data.csv is ignored and save.json is read.
